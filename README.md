# taximetro-moto-carro
Aplicativo PWA de taxímetro para moto ou carro. O motorista escolhe o tipo de veículo nas configurações. Funcionalidades: GPS offline, cálculo de tarifas por período (dia/noite/fim de semana), PIX configurável, link dinâmico para passageiro calcular valor e solicitar corrida via WhatsApp. Instalável como app na tela inicial.
